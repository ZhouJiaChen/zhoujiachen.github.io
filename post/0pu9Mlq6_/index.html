Level.1 - 开发环境

2020-02-25 / 2 min read

工欲善其事，必先利其器。

一、软件篇

IDEA
现代化的开发工具，可以提高开发效率😉。

Iterm2
替代系统自带的Terminal，也可以考虑使用Hyper。

Visual Studio Code
巨硬出品，代替sublime处理一些文本。

Docker
随时随地跑个服务。

Sourcetree
偶尔也想看看自己提交的graph🤡，或者考虑fork

TablePlus
一个简洁的数据库客户端，mysql可以选择Sequel Pro

Paw
替代品Postman

Paste
剪贴板管理，真的好用👍

Moom
窗口管理

Microsoft To Do
巨硬大法好

Mark Text
markdown编辑器

Gridea
一个静态博客的客户端

FinderGo
快速打开terminal到当前目录

Alfred 4
免费的功能已经很好用了，有需求可以买个Powerpack

Dash
遇到不会的，翻翻文档

Karabiner
如果你有把外置键盘压在Macbook键盘上的坏习惯，你可以使用此软件把Macbook的键盘禁用。当然，还有改键和更多其他强大的功能。

The Unarchiver
MacPaw出品，OSX平台好用到不行的压缩软件。

BetterTouchTool
没有Trackpad用户的福音，增强你的鼠标操作。

二、终端篇

zsh

chsh -s /bin/zsh

oh-my-zsh

sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

Homebrew
需要安装Command Line Tools，安装过程中会提示下载

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

sdkman

curl -s "https://get.sdkman.io" | bash

git-flow-avh
推荐使用homebrew安装。gitflow相关文档

cz-cli
格式化commit message，AngularJS 建议的提交格式
